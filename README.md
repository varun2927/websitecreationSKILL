# 🌐 websitecreationSKILL

A Claude skill that runs a **structured discovery interview** before building any website, web app, landing page, or digital product.

## What It Does

Instead of jumping straight into code, this skill guides Claude to ask the right questions first — extracting every real requirement a developer or designer would need. The result is a clean **Discovery Brief** that becomes the single source of truth for the entire build.

## When It Triggers

- "I want a website"
- "Build me an app"
- "Create a landing page for my startup"
- "I need an online presence"
- "Make me a portfolio"
- Any request to create, build, design, or launch a digital product

## What It Covers

| Phase | Topics |
|-------|--------|
| 1 — Purpose & Problem | Why it exists, who it's for, primary CTA |
| 2 — Pages & Structure | Site map, navigation, auth, admin panel |
| 3 — Features | Content, user actions, commerce, bookings, integrations, AI |
| 4 — Look & Feel | Design direction, colors, typography, animation |
| 5 — Content | Copy ownership, existing assets, legal pages |
| 6 — Technical | Stack, hosting, domain, compliance, maintenance |
| 7 — SEO & Performance | Keywords, load time, geo audience, Core Web Vitals |
| 8 — Launch & Beyond | Deadline, MVP scope, budget, success metrics |

## Output

A formatted **Discovery Summary Document** covering all of the above — ready to hand to any developer or designer to start building immediately.

## How to Use

Add `SKILL.md` to your Claude skills directory. Claude will automatically run this discovery flow whenever a website or app build is requested.
